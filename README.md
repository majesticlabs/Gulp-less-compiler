# Getting Started

## Feel free to use!
It would be very nice to leave me feedback or create pull-requests.

### Install npm
*Gulp 4.0.0 is required!*

```sh
npm install
```

### Start Gulp watch process

```sh
gulp
```

### Make your changes
*If you have made your changes, stop the "gulp watch"-process.*

```sh
ctrl + c
```
